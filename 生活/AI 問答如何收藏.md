# AI 的回覆，如果想要收藏，該如何處理?

>[!hint] 前言
>當我們想複製 AI 的回應內容時
>以 gemini 及 chatGPT 來說，複製回應的內容，都是一種 "markdown" 的格式。
  如果做成文件，其附檔名會是 "*.md"

- 如果是直接貼在記事本上，就會看到這樣的內容。可是這樣的內容很雜亂，怎麼辦?	![[Pasted image 20260423112050.png]]
- 有幾種方法可以處理
	- 複雜一點的
		把它存成 "*.md" 的檔案，並使用 vs code 這個編輯器，裝上 markdown 外掛。
	- 簡單一點的
		1. 裝 obsidian (windows 版本) [下載](https://github.com/obsidianmd/obsidian-releases/releases/download/v1.12.7/Obsidian-1.12.7.exe)
		2. 它的介面蠻明確的
		3. 你可以建立資料夾，做一些分類目錄。
		   ![[Pasted image 20260423112807.png]]
		
		4.  新建一個筆記
			![[Pasted image 20260423112736.png]]
		
		5. 把複製的內容，貼上筆記。就可以看到格式化後的內容。
		   ![[Pasted image 20260423113103.png]]
