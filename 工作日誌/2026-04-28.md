剛才您問的兩個問題，以下是回覆

Q1. 一個 api 可以同時處理 json 資料及 檔案嗎?
ans: 可以。

```java
@PutMapping("/api/project/status")
public Result updateProjectWithFile(
    @RequestParam("param") String jsonParam,  // JSON 字串
    @RequestParam("files") MultipartFile file  // 檔案
) {
    // 解析 JSON
    ProjectStatusRequest request = objectMapper.readValue(jsonParam, ProjectStatusRequest.class);
    
    // 處理檔案
    byte[] fileContent = file.getBytes();
    
    return projectService.updateStatus(request, fileContent);
}
```


Q2: 像這種 status 的內容，後端資料庫會怎麼存?
ans: 會用數字去存，在程式裡用 Enum 的方式去轉換。
```json
{
    "projectId": "PRJ123",
    "status": "suspend", // 狀態，暫停(suspend)、重啟(restart)、結案(finished)、終止(termination),
    "datetime": "2022-03-24 12:00",
    "content": "大事記內容",
}
```
