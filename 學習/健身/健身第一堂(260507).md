今天做的項目
1. 坐姿划船 (男廁門口) 10X4
2. 背部下拉 (第二排第二台) 15KGX10X4
 3. 坐姿划船 (窄) 20KGX10X4
 4. 啞鈴肩推. 1KGX10X4

(一) 坐姿划船 (男廁門口) 
1. 這個很重，在3F 處，有點難拿，所以用左腳踩一下輔助，左手拉到桿、右手順勢拉到。
2. 挺胸、手平行抓後，稍內旋(手肘稍往外..沈肩): 姿勢正確後，再拉。
3. 拉的時候吸氣、拉到與肩平行的位置。
4. 放的時候吐氣。
5. 一組10下，做四組。
   *4322 Row 坐式划船機: 這個沒有重量之分，本身就有一點重，姿勢一定要正確*

(二) 背部下拉- 1MTH002 LAT Pulldown Circular 高拉滑輪訓練機
1. 調整座高，雙腳平踩地面，調整腿墊位置，緊靠大腿墊。
2. 背部挺直、握住把手、肩與頸放鬆，以闊背肌出力下拉。
3. 拉至手肘靠近身體兩側。
4. 用力時吐、回復吸.
5. 15KG，10X4

(三) 坐姿划船(窄) - 背部划船
1. 腳踩踏板、手握握把、挺胸、微往後
2. 肩頰後收不聳肩，用背肌發力，向後拉
3. 不要憋氣避免受傷。
4. 20KG，10X4

(四) 啞鈴肩推. 1KGX10X4
1. 後來跑去一個很簡單的地方，左右手各拿了一個 1KG 的啞鈴
2. 一樣挺胸、沈肩、兩側手肘先彎曲，與地面直，
3. 往上伸展，再回復。
4. 1KG，10X4
